# fragments
Fragments back-end API

## Getting Started

You should have Node.js and npm installed on your computer.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AliOmarTO/fragments.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Scripts

Here are all the instructions on how to run the various scripts
* To lint the project:
  ```sh
  npm run lint
  ```
* To run the server normally:
  ```sh
  npm start
  ```
* To run in dev mode:
  ```sh
  npm run dev
  ```
* To run in debug mode:
  ```sh
  npm run debug
  ```
